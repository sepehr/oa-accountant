Description
===========
Simple accounting feature for the OpenAtrium distribution.

Disclaimer
==========
This module is in an early development stage, It is not intended to be used in production.

Dependencies
============
  - context
  - date
  - features
  - link
  - menu
  - nodeformcols
  - nodereference
  - number
  - optionwidgets
  - spaces
  - strongarm
  - text
  - userreference
  - views
  - views_calc

Installation
============
  - Goto admin/build/features path.
  - Enable "Atrium Accountant" under features section.
